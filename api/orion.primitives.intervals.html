orion.primitives.intervals module¶

orion.primitives.intervals.build_anomaly_intervals(X, y, time_column, severity=True, indices=False)[source]¶

Group together consecutive anomalous samples in anomaly intervals.

This is a dummy boundary detection function that groups together samples that have been consecutively flagged as anomalous and returns boundaries of anomalous intervals.

Optionally, it computes the average severity of each interval.

This detector is here only to serve as reference of what an boundary detection primitive looks like, and is not intended to be used in real scenarios.
