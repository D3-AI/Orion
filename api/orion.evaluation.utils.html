orion.evaluation.utils module¶

orion.evaluation.utils.from_list_points_labels(labels)[source]¶

Convert list of labels to list of tuples.

Convert a list of labels to a list of tuples marking the start and end interval of anomalies by defining a dummy timestamp range for usage.

Parameters

labels (list) – contains binary labels [0, 1].

Returns

tuple (start, end) timestamp.

Return type

list

orion.evaluation.utils.from_list_points_timestamps(timestamps, gap=1)[source]¶

Convert list of timestamps to list of tuples.

Convert a list of anomalies identified by timestamps, to a list of tuples marking the start and end interval of anomalies; make it contextually defined.

Parameters

  • timestamps (list) – contains timestamp of anomalies.

  • gap (int) – allowed gap between anomalies.

Returns

tuple (start, end) timestamp.

Return type

list
